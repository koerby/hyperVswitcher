Was kann HyperTool?

Alles, was du für
Hyper-V brauchst.

Von VM-Lifecycle bis USB-Forwarding – HyperTool bringt alle essenziellen Hyper-V-Workflows in eine aufgeräumte, moderne Oberfläche.

🖥

VM-Management

Start, Stop, Hard Off, Restart und Konsolenzugriff für alle Hyper-V-VMs – direkt aus der App, ohne PowerShell.

Start / Stop Hard Off Konsole Default-VM
🌐

Netzwerk-Kontrolle

Adaptergenaues Switch-Handling auch für Multi-NIC-Setups. Gateway und Default Switch werden farbcodiert angezeigt.

Switch-Handling Multi-NIC Gateway-Status
📸

Snapshot-Manager

Baumdarstellung aller Snapshots mit Restore, Delete und Create – inklusive Hierarchieansicht für verschachtelte Snapshots.

Baumansicht Restore Create Delete
🔌

USB-Forwarding (Host)

USB-Geräte per usbipd-win mit einem Klick teilen oder freigeben. Der Tray zeigt den Dienststatus in Echtzeit.

usbipd-win Share / Unshare Tray-Status
🔌

USB-Client (Guest)

Geräte vom Host laden und per Hyper-V Socket oder IP-Mode verbinden. Transport-Chips zeigen den Modus direkt an.

Hyper-V Socket IP-Mode usbip-win2
📁

Shared Folders

Host-Katalog und Freigaben im Guest per WinFsp mounten. Laufwerkszuordnungen im Explorer sichtbar – ohne klassisches SMB.

WinFsp hypertool-file Explorer-sichtbar
🎨

Dark & Light Theme

Vollständige Dark/Light-Unterstützung durch WinUI 3. Theme-Wechsel mit kurzem Reload-Screen ohne Datenverlust.

Dark Mode Light Mode WinUI 3
🔄

Auto-Updates

In-App Updatecheck gegen GitHub Releases. Download und Installer-Start direkt aus der App heraus.

GitHub Releases In-App Update
🗂

Tray Control Center

Kompaktes Tray-Menü mit Schnellaktionen, Runtime-Status und Installationsbutton bei fehlenden Abhängigkeiten.

Tray-Menü Schnellaktionen Runtime-Status
App-Vorschau

Schau hinter die Kulissen.

Jede Ansicht ist auf maximale Übersichtlichkeit optimiert – von den VM-Chips bis zum Snapshot-Baum.

Host

Hyper-V im Griff –
auf dem Host.

HyperTool Host ist deine Kommandozentrale für alle Hyper-V-Aktionen. Kein nerviges Öffnen des Hyper-V-Managers – starte, stoppe und snapshot-e deine VMs in Sekunden.

  • ✓ VM-Lifecycle: Start, Stop, Hard Off, Restart, Konsole
  • ✓ Snapshot-Baum mit hierarchischer Darstellung
  • ✓ Netzwerk-Adapter mit Switch-Zuweisung
  • ✓ USB-Sharing via usbipd-win per Knopfdruck
  • ✓ Tray Control Center mit Dienststatus
  • ✓ Shared-Folder-Katalog für Guest-Mounts
  • ✓ In-App Update-Check & Installer

Guest

Im Guest? Alles
unter Kontrolle.

HyperTool Guest läuft direkt in deiner Windows-VM und verbindet sich mit dem Host. USB-Geräte durchreichen, Shared Folders mounten – alles ohne Adminrechte-Gefrickel.

  • ✓ USB-Geräte vom Host laden & verbinden
  • ✓ Transport: Hyper-V Socket oder IP-Mode
  • ✓ Shared Folders per WinFsp mounten
  • ✓ Mount-Status überwachen
  • ✓ Tray Control Center mit Schnellaktionen
  • ✓ Autostart & Minimize-to-Tray
  • ✓ Theme-Neustart erhält die Menüseite
